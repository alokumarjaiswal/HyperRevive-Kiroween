# HyperRevive: Resurrecting Apple's HyperCard for Kiroween

A modern web-based revival of HyperCard using MERN stack and Kiro AI, focusing on card-based knowledge bases for creators. Category: Resurrection.

## MVP Features (Revived)
- Card creation and stack navigation (linked, editable cards with text/images).
- Button interactions (e.g., next/previous card).
- Field editing (dynamic text/image inputs).

## Modern Updates
- Kiro-generated scripting (vibe coding for simple logic).
- Dynamic API content (e.g., embed live data on cards).

## Use Case
Interactive personal knowledge base (e.g., storyboarding for writers).

## Tech Stack
MERN (MongoDB, Express, React, Node.js); Kiro for AI-assisted development.

## Installation
- Clone repo: git clone https://github.com/alokumarjaiswal/HyperRevive-Kiroween.git
- Install dependencies: npm install
- Run: npm start (frontend) / node server.js (backend)
- For Kiro: See /.kiro for specs and prompts.
